## Core color palette

- **Primary brand**: #3DC4C4 (main teal)  
  - Use for: buttons, key icons, section highlights, progress bars. [colorhunt](https://colorhunt.co/palettes/teal)
- Primary dark (text / accents): #102A3A (navy‑charcoal)  
  - Use for: main headings, body text on white, some backgrounds to anchor trust. [visme](https://visme.co/blog/website-color-schemes/)

  - Use as main page background for cleanliness and readability. [visme](https://visme.co/blog/website-color-schemes/)

  - Use for alternate section backgrounds, cards, and FAQ blocks.

  - Use for card borders, table lines, and light dividers.

Optional accent (very subtle):
- Soft accent: #FFB872 (warm soft orange) for tiny highlights like icons or hover states (max 5–10% of UI). [bungalowwebdesign](https://bungalowwebdesign.com/9-best-color-palettes-for-accountant-websites-in-2024-examples/)

## Layout and section styling

### Header & navigation

- Background: white (#FFFFFF).  
- Logo: “Jim Harvey” in dark navy (#102A3A), small underline or icon in #3DC4C4.  
- Nav links: #102A3A default, #3DC4C4 on hover with bottom border 2px #3DC4C4.  
- Primary header button:
  - Background: #3DC4C4
  - Text: white
  - Border radius: 6–8px
  - Hover: slightly darker teal (#35AFAF). [ventnorwebagency](https://ventnorwebagency.com/ui-ux-design-for-accounting-websites/)

Sticky header on scroll with subtle shadow (0 2px 8px rgba(0,0,0,0.06)) to keep CTA visible. [theedigital](https://www.theedigital.com/blog/makes-great-website-cpa)

### Typography & spacing

- Heading font: Clean, modern sans (e.g., Poppins, Inter, or similar).  
- Body font: Inter / Roboto / system sans for legibility.  
- Colors:
  - Body text: #102A3A on white
  - Muted text: #6E7A89 for secondary copy. [theedigital](https://www.theedigital.com/blog/makes-great-website-cpa)

Spacing:
- Large vertical padding per section: 80–100px desktop, 48–64px mobile.  
- Max content width: 1120–1200px for comfortable reading. [theedigital](https://www.theedigital.com/blog/makes-great-website-cpa)

***

### Quick “style rules” to follow

- Use teal (#3DC4C4) mainly for:
  - Buttons
  - Icons
  - Small highlights and lines  
  Avoid using it for large text blocks to keep contrast strong. [piktochart](https://piktochart.com/tips/teal-color-palette)
- Keep most backgrounds white or very light gray to feel clean and financial.  
- Limit accent color (orange) to small touches only, if you even use it.  
- Make sure body text is dark navy on white for maximum readability. [visme](https://visme.co/blog/website-color-schemes/)
